# kindred-emotion-extension

follow these instructions to load the extension: 

0. download the zip, then uncompress the zipped file
1. go `chrome://extensions`
2. turn on developer mode (it should be in the top right hand corner)
3. click `load unpacked` and select the folder containing the uncompressed file
4. extension should now appear in your Chrome toolbar. you might have to click on the puzzle icon to pin it
5. click on the icon to access the extension and try out the diff visualizations as you go about doing your work
6. give feedback by clicking on `take survey`
